# Heluming Personal Website

Static single-page personal brand website for Heluming, an AI visual creator.

## Local Preview

Run this inside the `site` directory:

```bash
npm run start
```

Then open:

```text
http://127.0.0.1:4173
```

## Public Display Checklist
